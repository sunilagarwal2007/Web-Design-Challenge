Summary: Latitude vs. Cloudiness

In this project I created a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. To create this representative model of weather across world cities, I utilized tools such as Pandas, Jupyter Notebooks, and the OpenWeatherMap API.

I then built a series of scatter plots using Matplotlib to showcase the relationship between the latitude and factors such as temperature, humidity, cloudiness, and wind speed. This site provides the source data used for this project along with the 4 plots that I created.
